# Quick Start Guide

## Installation & Running

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm start
```

## Test the System

1. Go to `http://localhost:3000`
2. Sign up as different roles (Admin, Teacher, Student)
3. Login and explore features
